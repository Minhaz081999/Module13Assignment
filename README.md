# BMI Calculator App

A simple BMI calculator built with Flutter. It allows users to input weight and height in multiple units and get a color-coded BMI result.

## Features
- Weight: kg or pounds
- Height: cm, feet/inch, or meters
- Auto unit conversion
- Color-coded BMI category
# BMI Calculation

Formula (Metric): BMI = weight_kg / (height_m)^2

Show the BMI value to 1 decimal place (e.g., 24.2).
# Conversions

kg = lb × 0.45359237

m = cm / 100

m = (feet × 12 + inch) × 0.0254
# Display BMI score and category:

Underweight (< 18.5)

Normal (18.5 – 24.9)

Overweight (25.0 – 29.9)

Obese (≥ 30.0)
# Color based on category:

Underweight → Blue

Normal → Green

Overweight → Orange

Obese → Red

## How to Run
1. Clone the repo
2. Run `flutter pub get`
3. Run `flutter run`
