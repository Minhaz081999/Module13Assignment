# BMI Calculator App

A simple BMI calculator built with Flutter. It allows users to input weight and height in multiple units and get a color-coded BMI result.

## Features
- Weight: kg or pounds
- Height: cm, feet/inch, or meters
- Auto unit conversion
- Color-coded BMI category

## How to Run
1. Clone the repo
2. Run `flutter pub get`
3. Run `flutter run`
